# CHIP-8 Emulator

## English

### About
A CHIP-8 emulator implementation that faithfully reproduces the behavior of the original 1970s virtual machine. This project demonstrates core emulation concepts including instruction decoding, memory management, and graphics rendering.

### Features
- Full CHIP-8 instruction set support
- Display rendering
- Keyboard input handling
- Sound support

### Getting Started
```bash
git clone <repository>
cd Chip8-Emu
make
./chip8-emu [rom_file]
```

### Technical Details
- **Language**: C/C++
- **Architecture**: 4KB memory, 16-bit opcodes
- **Display**: 64x32 pixels

## Português

### Sobre
Uma implementação de emulador CHIP-8 que reproduz fielmente o comportamento da máquina virtual original dos anos 1970. Este projeto demonstra conceitos fundamentais de emulação incluindo decodificação de instruções, gerenciamento de memória e renderização gráfica.

### Funcionalidades
- Suporte completo ao conjunto de instruções CHIP-8
- Renderização de display
- Tratamento de entrada de teclado
- Suporte a som

### Começando
```bash
git clone <repositório>
cd Chip8-Emu
make
./chip8-emu [arquivo_rom]
```

### Detalhes Técnicos
- **Linguagem**: C/C++
- **Arquitetura**: 4KB memória, opcodes de 16 bits
- **Display**: 64x32 pixels
